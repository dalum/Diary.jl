configure(; kwargs...)

Change configuration for Diary.jl in the current session.

Supported keywords

  • author: Author name put in the header. (default: "")
  • date_format: Dates.formatâ€“compatible date format used in the header. (default: "E U d HH:MM")
  • diary_file_name: Default name of the diary file. (default: "diary.jl")

DiaryConfig

Mutable configuration for Diary.jl. Diary.jl contains a single global instance of this type, GLOBAL_CONFIG. Configuration of Diary.jl should be done through configure rather than direct manipulations of GLOBAL_CONFIG.
